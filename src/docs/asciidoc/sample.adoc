== PlantUMLによるクラス図の例
UMLはPlantUMLで記述できる。日本語（UTF-8）でも記述可能。

.クラス図
[plantuml, classediagram, png]
....
class "クラスA" {
  +メソッド1()
}
class "クラスB" {
  +メソッド2()
}
class "クラスC" {
}
"クラスA" -- "クラスB"
"クラスA" <|-- "クラスC"
....

== Ditaaによる図の例
箱と線で表現するものはDitaaで記述可能。日本語（UTF-8）は文字化けする

.a = 1 + 3の構文木
[ditaa, syntaxtree]
....
    /--------\
    |cmpExpr |
    |text:'='|
    \--------/
       |   |
  /----/   \-----\
  |              |
/--------\   /--------\
|variable|   |addExpr |
|text:'a'|   |text:'+'|
\--------/   \--------/
               |   |
           /---/   \---\
           |           |
     /----------\  /----------\
     |literal   |  |literal   |
     |text:'1'  |  |text:'3'  |
     \----------/  \----------/
....
